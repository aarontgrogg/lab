  • Glossary
    Component
    Reusable HTML, could be button, widget, module, etc.
    term
    def
  • Primer
    • App framework, similar to React/Vue
    • Biggest diff is Svelte is a compiler, shipping ONLY what is needed to the client
    • Can create full app or mere component
    • Much faster, and smaller, than similar frameworks
    • Create JS as components, export/import into .svelte files
  • Insert Variables into HTML using `{}` 
    // declare variables
let src = 'logo.jpg';
// then insert into HTML
<img src="{src}" alt="Logo">
// if attribute and variable name are the same, can use shortcuts
let src    = 'logo.jpg';
let width  = 100;
let height = 100;
let alt    = 'Logo';
// insert variables without attribute name, = or ""
<img {src} {width} {height} {alt}>
// output:
<img src="logo.jpg" width="100" height="100" alt="Logo">

// NOTE: if your string has HTML, you need to encapsulate it within `{@html xxxxx}``
let str = 'Hello <strong>World</strong>!';
<p>{@html str}</p>
// NOTE: Svelte performs NO sanitation on imported data, so escape yourself to prevent XSS
  • Insert styles right into component, output is scoped 
    // add `style`right into component
<style>
    #title { color: purple; }
    p { color: red; }
</style>
<h1 id="title">
	Title
</h1>
<p>
    Hello World!
</p>
// output is scoped with component-level hash; '4uj140' below
<style id="svelte-4uj140">
    #title.svelte-4uj140{color:purple}
    p.svelte-4uj140{color:red}
</style>
<h1 class="svelte-4uj140">This is a paragraph.</h1>
<p class="svelte-4uj140">This is a paragraph.</p>

// NOTE: Svelte will actually warn you about any unused CSS, so you know it is safe to remove!
  • Import/Export components 
    // import a component
<script>
    import ComponentName from './Component/Path/Filename.svelte';
</script>
// and consume
<p>Hello World!</p>
<ComponentName/>

// NOTE: imported components are neither affected by local CSS, nor does their CSS affect this component
// NOTE: the convention is to Capitalize component names, to help differentiate from native HTML elements
